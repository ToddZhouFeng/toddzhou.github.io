# 为什么要搭博客？

​	虽然现在比较流行的是朋友圈、公众号，但这些内容都太散乱。有一个博客则可以很好的整理文章。而且博客可以自己定制，可玩性也比较高。

# 如何用Github搭博客

首先申请Github账号，然后创建一个“账号名+github.io”的repo，然后设置里就会出现相关设置，按它的步骤写就行了。具体步骤请看[傻瓜都可以利用github pages建博客](http://cyzus.github.io/2015/06/21/github-build-blog/)，我就是照着上面来做的。

然后有一点特别操蛋的是，下载了jekyll主题后，你需要一个一个修改页面上的文字，有时候根本找不到要修改的文件，这个花了我一个中午的时间去找......
