# Git历史

&emsp;&emsp;Git 的诞生和 Linux 系统密切相关，Linux 开发它就是为了 Linux 这种大规模的项目服务的，这就使得它有着很多优点，比如：
* **免费**
* **快速**
* **简单**
* 分布式
* 多分支

&emsp;&emsp;学会用 Git 可以帮助我们更好地进行代码管理。
