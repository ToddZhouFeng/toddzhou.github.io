# Git历史

&emsp;&emsp;Git 的诞生和 Linux 系统密切相关，其开发伊始就是为了 Linux 这种大规模的项目服务的，这就使得它有着很多优点，比如：

* 快速
* 简单
* 分布式
* 多分支

&emsp;&emsp;如今，git 以及 github、gitlab等 已经成为了大部分码农管理代码的首选。
